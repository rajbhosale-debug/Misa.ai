# MISA.AI MSIX Package Assets

This directory contains visual assets for the MISA.AI MSIX package.

## Required Assets

The following assets should be placed in this directory before building the MSIX package:

### Application Icons
- `StoreLogo.png` - 50x50 pixels - Windows Store listing
- `SmallTile.png` - 71x71 pixels - Small tile on Start screen
- `Square150x150Logo.png` - 150x150 pixels - Medium tile on Start screen
- `Square44x44Logo.png` - 44x44 pixels - Application icon
- `Wide310x150Logo.png` - 310x150 pixels - Wide tile on Start screen
- `LargeTile.png` - 310x310 pixels - Large tile on Start screen

### Screenshots
- `Screenshot1.png` - 1366x768 pixels - Store screenshot 1
- `Screenshot2.png` - 1366x768 pixels - Store screenshot 2
- `Screenshot3.png` - 1366x768 pixels - Store screenshot 3
- `Screenshot4.png` - 1366x768 pixels - Store screenshot 4
- `Screenshot5.png` - 1366x768 pixels - Store screenshot 5

### Splash Screen
- `SplashScreen.png` - 620x300 pixels - Application splash screen

## Asset Requirements

- Format: PNG
- Color depth: 32-bit (alpha channel supported)
- Scale factors: 100%, 125%, 150%, 200%, 400% (if multiple sizes are provided)
- Transparency: Supported for logos with alpha channel

## Temporary Placeholders

Currently, placeholder README files exist for required assets. These should be replaced with actual application graphics before release.

## Design Guidelines

Follow Microsoft's design guidelines for Windows Store applications:
- Use the MISA.AI brand colors and logo
- Maintain consistency with the desktop and web applications
- Ensure high contrast and readability
- Test on different screen sizes and scale factors
