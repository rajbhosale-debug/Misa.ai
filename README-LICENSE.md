## üöÄ **Quick Start**

### **For Personal Use**
```bash
git clone https://github.com/your-username/misa-ai.git
cd misa.ai
./scripts/install-dependencies.sh
./scripts/build-all.sh
./scripts/install.sh
misak-kernel --data-dir ~/.local/share/misa-ai
```

### **For Deployment**
```bash
# Start full stack
docker-compose up -d

# Install system-wide
sudo ./scripts/install.sh

# Start services
misa-kernel --data-dir /opt/misa-ai
http://localhost:8080
```
