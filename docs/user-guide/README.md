## üéØ Quick Start

1. **Install MISA.AI** (2-3 minutes):
   ```bash
   curl -fsSL https://raw.githubusercontent.com/misa-ai/misa.ai/main/scripts/install.sh | bash
   ```

2. **Complete Setup** (2 minutes):
   - Open http://localhost:3000
   - Follow the setup wizard
   - Configure privacy and security settings

3. **Start Using**:
   - Try voice commands: "Hey Misa, what's on my calendar today?"
   - Explore applications in the sidebar
   - Check out the built-in tutorials
